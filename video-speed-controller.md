## video speed controller tutorial

- **`shift+[`** - decrease speed 0.1x
- **`shift+]`** - increase speed 0.1x
- **`-`** - go back 180 second / 3 mnts.
- **`+`** - go forward 180 second / 3 mnts.

- **`shift+y`** - reset speed.(set speed into 1x)
- **`\`** - set speed to 1.75x
